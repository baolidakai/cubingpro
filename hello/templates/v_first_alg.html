V first / L4E (Last 4 Edges)

In V first method, we first solve one layer except for one edge, this is done intuitively.
Then we apply an algorithm to solve last 4 edges together. To reduce algorithm count, we allow the top layer to be misaligned with bottom layer.
Eventually, we do top layer AUF and tips.
This method is also known as L4E (last 4 edges).
We present all the L4E algorithms here.
